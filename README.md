# OIBSIP_web_development_task2
Portfolio Page
This is a personal portfolio website created using HTML and CSS.
It is designed to showcase a web developer’s profile, projects, skills, and contact details.
	•	The navbar helps users navigate different sections smoothly.
	•	The hero section introduces the developer with a background image and call-to-action buttons.
	•	The about section gives a brief introduction.
	•	The projects section displays completed projects using interactive cards.
	•	The skills section highlights technical skills.
	•	The contact section allows users to send messages.
	•	The website is responsive, animated, and user-friendly.
